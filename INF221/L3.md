## 1. Divide-and-Conquer Paradigm

Three steps at each level of recursion:
1. **Divide** the problem into subproblems
2. **Conquer** subproblems recursively (base case: solve directly)
3. **Combine** solutions into the original solution

> [!tip] Why it matters
> D&C algorithms like merge sort achieve $\Theta(n \lg n)$ worst-case, far better than insertion sort's $\Theta(n^2)$.

---

### Tromino Example
- **Input:** $2^n \times 2^n$ grid with 1 filled square
- **Goal:** tile the rest with L-shaped trominos
- **Algorithm:**
	1. Divide grid into 4 quadrants
	2. Place 1 tromino at the center covering 1 square in each of the 3 empty quadrants
	3. Now all 4 quadrants have exactly 1 filled square → recurse
	4. Base case: $2 \times 2$ grid
- **Tromino theorem:** $\forall\, n \in \mathbb{N}_0,\; \exists\, m \in \mathbb{N}_0 : 4^n = 3m + 1$

## 2. Merge Sort Runtime Analysis

### Pseudocode
```text
MERGE-SORT(A, p, r)
1: if p < r
2:     q = ⌊(p + r) / 2⌋
3:     MERGE-SORT(A, p, q)
4:     MERGE-SORT(A, q + 1, r)
5:     MERGE(A, p, q, r)
```

### General D&C Recurrence
$$T(n) = \begin{cases} \Theta(1) & \text{if } n \leq c \\ aT(n/b) + D(n) + C(n) & \text{otherwise} \end{cases}$$

| Component | Merge Sort Value |
| --------- | ---------------- |
| $a$ (subproblems) | 2 |
| $b$ (shrink factor) | 2 |
| $D(n)$ (divide) | $\Theta(1)$ |
| $C(n)$ (combine/merge) | $\Theta(n)$ |

### Merge Sort Recurrence
$$T(n) = \begin{cases} c & \text{if } n = 1 \\ 2T(n/2) + cn & \text{if } n > 1 \end{cases}$$

### Recursion Tree Solution
- $\lg n + 1$ levels, cost $cn$ per level
- **Total:** $cn(\lg n + 1) = cn \lg n + cn$
- **Result:** $T(n) = \Theta(n \lg n)$

## 3. Growth of Functions — Asymptotic Notation

### Θ-notation (tight bound)
$$\Theta(g(n)) = \{ f(n) \mid \exists\, c_1 > 0,\, c_2 > 0,\, n_0 \in \mathbb{N} : 0 \leq c_1 g(n) \leq f(n) \leq c_2 g(n) \;\forall\, n \geq n_0 \}$$
- $f(n)$ is **sandwiched** between $c_1 g(n)$ and $c_2 g(n)$

### O-notation (upper bound)
$$O(g(n)) = \{ f(n) \mid \exists\, c > 0,\, n_0 \in \mathbb{N} : 0 \leq f(n) \leq c\, g(n) \;\forall\, n \geq n_0 \}$$
- $f(n)$ grows **no faster** than $g(n)$

### Ω-notation (lower bound)
$$\Omega(g(n)) = \{ f(n) \mid \exists\, c > 0,\, n_0 \in \mathbb{N} : 0 \leq c\, g(n) \leq f(n) \;\forall\, n \geq n_0 \}$$
- $f(n)$ grows **at least as fast** as $g(n)$

### Key Theorem
$$f(n) = \Theta(g(n)) \iff f(n) = O(g(n)) \;\wedge\; f(n) = \Omega(g(n))$$

### Relationships
$$\Theta(g(n)) \subseteq O(g(n)), \qquad \Theta(g(n)) \subseteq \Omega(g(n))$$

### Properties
| Property | Applies to |
| -------- | ---------- |
| **Transitivity** | $\Theta, O, \Omega$ |
| **Reflexivity** | $\Theta, O, \Omega$ |
| **Symmetry** | $\Theta$ only |
| **Transpose symmetry** | $f = O(g) \iff g = \Omega(f)$ |

### Comparison Analogy
| $\leq$ | $=$ | $\geq$ |
| --- | --- | --- |
| $O$ | $\Theta$ | $\Omega$ |

### Quick Examples
| Function | $O$ | $\Theta$ | $\Omega$ |
| -------- | --- | -------- | -------- |
| $5n^2 + 2n - 10$ | $O(n^2)$ ✅ | $\Theta(n^2)$ ✅ | $\Omega(n^2)$ ✅ |
| $n$ | $O(n^2)$ ✅ | $\Theta(n^2)$ ❌ | $\Omega(n^2)$ ❌ |
| $10^{-6}n^3$ | $O(n^2)$ ❌ | — | $\Omega(n^2)$ ✅ |

### General Rule for Polynomials
$$p(n) = \sum_{i=0}^{d} a_i n^i, \quad a_d > 0 \implies p(n) = \Theta(n^d)$$

## 4. Solving Recurrences — Master Method

For recurrences of the form:
$$T(n) = aT(n/b) + f(n), \quad a \geq 1,\; b > 1,\; f(n) > 0$$

Compare $f(n)$ to $n^{\log_b a}$:

| Case | Condition | Solution |
| ---- | --------- | -------- |
| **1** | $f(n) = O(n^{\log_b a - \epsilon})$ for some $\epsilon > 0$ | $T(n) = \Theta(n^{\log_b a})$ |
| **2** | $f(n) = \Theta(n^{\log_b a})$ | $T(n) = \Theta(n^{\log_b a} \lg n)$ |
| **3** | $f(n) = \Omega(n^{\log_b a + \epsilon})$ for some $\epsilon > 0$ **+ regularity** | $T(n) = \Theta(f(n))$ |

> [!warning] Case 3 Regularity Condition
> Must also show $af(n/b) \leq cf(n)$ for some $c < 1$ and all large $n$.
> Always satisfied if $f(n) = n^k$.

### Worked Examples

| Recurrence | $a, b$ | $n^{\log_b a}$ | Case | $T(n)$ |
| ---------- | ------ | -------------- | ---- | ------ |
| $5T(n/2) + \Theta(n^2)$ | 5, 2 | $n^{\lg 5} \approx n^{2.32}$ | 1 | $\Theta(n^{\lg 5})$ |
| $27T(n/3) + \Theta(n^3)$ | 27, 3 | $n^3$ | 2 | $\Theta(n^3 \lg n)$ |
| $5T(n/2) + \Theta(n^3)$ | 5, 2 | $n^{\lg 5} \approx n^{2.32}$ | 3 | $\Theta(n^3)$ |

> [!danger] Master Theorem Does NOT Apply When
> - $a$ is not constant (e.g., $a = 2^n$)
> - Regularity condition fails (e.g., oscillating $f(n)$)

## 5. Useful Math Reference

### Floor & Ceiling
- $\lfloor x \rfloor$ = largest integer $\leq x$
- $\lceil x \rceil$ = smallest integer $\geq x$
- $x - 1 < \lfloor x \rfloor \leq x \leq \lceil x \rceil < x + 1$
- $\lfloor n/2 \rfloor + \lceil n/2 \rceil = n$

### Logarithm Identities
| Identity | |
| -------- | - |
| $\lg n = \log_2 n$ | binary log |
| $\log_b a = \frac{\lg a}{\lg b}$ | change of base |
| $\log_c(ab) = \log_c a + \log_c b$ | product rule |
| $\log_b a^n = n \log_b a$ | power rule |
| $a^{\log_b c} = c^{\log_b a}$ | exponent swap |

### Exponential Rules
- $(a^m)^n = a^{mn}$
- $a^m \cdot a^n = a^{m+n}$
- $e^x \geq 1 + x$
- $0^0 = 1$ (by definition)

### Factorials
- $n! = 1 \times 2 \times \cdots \times n$, $\quad 0! = 1$
- **Stirling:** $n! = \sqrt{2\pi n}\left(\frac{n}{e}\right)^n\left(1 + \Theta\left(\frac{1}{n}\right)\right)$
- $\lg n! = \Theta(n \lg n)$

### Common Sums
| Sum | Result |
| --- | ------ |
| $\sum_{k=1}^{n} k$ | $\frac{n(n+1)}{2}$ |
| $\sum_{k=1}^{n} k^2$ | $\frac{n(n+1)(2n+1)}{6}$ |
| $\sum_{k=1}^{n} k^3$ | $\frac{n^2(n+1)^2}{4}$ |
| $\sum_{k=0}^{n} x^k$ ($x \neq 1$) | $\frac{x^{n+1} - 1}{x - 1}$ |
| $\sum_{k=0}^{\infty} x^k$ ($\|x\| < 1$) | $\frac{1}{1 - x}$ |

### Modulo
$$a \bmod n = a - n\left\lfloor \frac{a}{n} \right\rfloor$$
